
# Интернет-Радио на PyQt6 с использованием библиотеки Python-VLC

## Описание проекта

Данное приложение представляет собой интернет-радио, разработанное с использованием **PyQt6** и библиотеки **python-vlc**. Пользователи могут легко прослушивать различные радиопотоки через удобный графический интерфейс.

---

## Установка

### Требования

Для успешной работы приложения необходимо установить следующие библиотеки:

- Python 3.x
- PyQt6
- python-vlc

### Установка зависимостей

Для установки необходимых библиотек выполните следующую команду:

```bash
pip install -r requirements.txt
```

## Клонирование репозитория
Сначала клонируйте репозиторий проекта:

```bash
git clone https://github.com/graal101/Radio.git
cd ваш_репозиторий
```
---

## Запуск приложения
Для запуска приложения выполните следующую команду:

```bash
python main_radio.py
```

---

## Использование

1. **Укажите URL-адрес радиопотока**: Введите адрес радиостанции в текстовое поле.
2. **Запустите воспроизведение**: Нажмите кнопку "Play".
3. **Управление воспроизведением**:
   - Нажмите "Pause" для приостановки воспроизведения.
   - Нажмите "Stop" для остановки воспроизведения.
